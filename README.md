

## 说明

支持平台：安卓（4.4版本以上）

### 运行前准备工作
1、下载nodejs [参考](https://nodejs.org/en/)

2、安装ionic [参考](https://ionicframework.com/)

```bash
$ npm install -g ionic@latest
```

### 使用说明

1、下载文件，根目录运行

```bash
$ npm install
```

2、添加Android平台

```bash
$ ionic cordova platform add android
```

3、电脑接入Android手机，运行

```bash
$ ionic cordova run android
```
